1. Deskripsi Sistem

Sistem ini dirancang untuk mengontrol pergerakan dua silinder pneumatik menggunakan PLC, solenoid, dan sensor metal. Cylinder 1 dan Cylinder 2 bekerja secara terkoordinasi untuk menyelesaikan proses dengan pengaturan waktu tertentu.

2. Komponen Utama

  • PLC: 1. PLC (Programmable Logic Controller) Fungsi: Mengontrol seluruh sistem berdasarkan logika yang diprogram. Model: [Isi model PLC yang digunakan].
  • Solenoid Valve: Solenoid 1: Mengontrol pergerakan Cylinder 1. Solenoid 2: Mengontrol pergerakan Cylinder 2.
  • Silinder Pneumatik: Cylinder 1: Bergerak maju untuk memulai proses. Cylinder 2: Bergerak maju dan mundur sesuai perintah PLC.
  • Sensor Metal: Fungsi: Mendeteksi posisi Cylinder 1 untuk mengaktifkan langkah berikutnya.
  • Power Supply: Fungsi: Menyediakan daya untuk PLC dan solenoid
  • Timer Internal PLC: Fungsi: Mengatur jeda waktu antar langkah.

3. Cara Kerja Sistem

  1. Aktivasi awal: Saat sistem diaktifkan, PLC mengirimkan sinyal HIGH ke Solenoid 1 (misalnya, melalui pin output Q0.0) untuk menggerakkan Cylinder 1 maju.
  2. Trigger sensor: Cylinder 1 bergerak maju hingga memicu sensor metal. Sensor metal mengirimkan sinyal ke PLC melalui pin input (misalnya, I0.1).
  3. Aktivasi Cylinder 2: Setelah menerima sinyal dari sensor metal, PLC mengaktifkan Solenoid 2 (pin Q0.1) dengan jeda waktu 3 detik untuk menggerakkan Cylinder 2 maju.
  4. Gerakan Cylinder 2: Cylinder 2 bergerak maju selama 5 detik. Setelah itu, Cylinder 2 mundur selama 5 detik.
  5. Reset Cylinder 1: Setelah Cylinder 2 selesai bergerak, PLC mengatur Cylinder 1 untuk kembali ke posisi awal.

4. Wiring Diagram

  1. Solenoid 1 Pin output PLC Q0.0 → Solenoid 1 → Power Supply.
  2. Solenoid 2 Pin output PLC Q0.1 → Solenoid 2 → Power Supply.
  3. Sensor Metal Pin input PLC I0.1 → Sensor Metal → Power Supply.
  4. Power Supply Daya 24V untuk PLC dan solenoid.
  5. Timer Internal Dikontrol oleh program PLC.

5. Pemrograman PLC

Program PLC dirancang menggunakan ladder diagram dengan langkah-langkah berikut:

  1. Kirim sinyal HIGH ke Q0.0 untuk mengaktifkan Solenoid 1.
  2. Tunggu sinyal dari I0.1 (sensor metal).
  3. Aktifkan Q0.1 (Solenoid 2) dengan jeda waktu 3 detik menggunakan timer.
  4. Atur timer untuk menggerakkan Cylinder 2 maju selama 5 detik dan mundur selama 5 detik.
  5. Reset Cylinder 1 ke posisi awal dengan mematikan Q0.0.

6. Troubleshooting

  • Cylinder tidak bergerak: Periksa koneksi wiring dan power supply.
  • Sensor tidak mendeteksi: Pastikan sensor metal terpasang dengan benar.
  • Timer tidak bekerja: Periksa program PLC untuk pengaturan waktu.
